\section{Research Objectives and Questions}

The primary overarching objective of this research is to conceptualize, design, and theoretically evaluate \textbf{Deltx}, a novel architecture integrating static code analysis, AI detection, and predictive modeling. To systematically address the gaps identified in the literature, this project is decomposed into the following specific research objectives:

\subsection{Research Objectives}

\begin{enumerate}
    
    \item \textbf{AI Code Detection Methodology}
    
    To define and operationalize an advanced \textbf{AI Code Detection} methodology that utilizes structural unpredictability, entropy variance, and stylometric feature distributions \parencite{beyond_perplexity_character_distribution_signatures_mdta} to accurately estimate the proportion of AI-generated code within a repository commit.
    
    \item \textbf{Quality Mapping and Weighting Mechanism}
    
    To formulate a mapping and weighting mechanism that translates granular SonarQube rule violations \parencite{sonarqube_metrics_definition} into four primary \textbf{ISO/IEC 25010 dimensions} \parencite{isoiec25010_quality_model_emergentmind} (Maintainability, Correctness, Security, Efficiency), explicitly factoring in issue severity, occurrence frequency, code churn \parencite{relative_code_churn_defect_density}, and call-graph centrality \parencite{centrality_in_networks_finding_most_important_nodes}.
    
    \item \textbf{Conflict-Resistant Quality Scoring System}
    
    To engineer a conflict-resistant \textbf{Quality Scoring System}, inspired by hierarchical frameworks like the Squale model \parencite{squale_model_practice_based_industrial_quality_model, computing_weighted_average_in_squale}, that computes normalized scores on a 0--100 scale \parencite{min_max_and_z_score_normalization, normalize_data_between_0_and_100} for each overarching quality dimension, with explicit penalization of outliers \parencite{mord11a_continuous_weighted_metric_aggregation}.
    
    \item \textbf{PatchTST-Based Predictive Modeling Engine}
    
    To architect and formalize a \textbf{PatchTST-based Predictive Modeling Engine} \parencite{decompatchtst_hybrid_framework, patchtst_nixtla} capable of ingesting multivariate historical metric time-series to forecast future quality score trajectories \parencite{comparative_analysis_lstm_gru_transformer_construction_cost}, complete with statistically robust confidence intervals \parencite{fault_prediction_software_metrics_sonarqube_rules}.
    
    \item \textbf{Interpretation Layer with SHAP Integration}
    
    To integrate a comprehensive \textbf{Interpretation Layer} utilizing SHAP methodologies \parencite{perspective_on_xai_methods_shap_and_lime, comparative_analysis_lime_shap_interpreters_diabetes} to translate numerical quality forecasts into semantic, human-readable insights \parencite{explainability_testing_ai_shap_lime_testriq}, benchmarked against empirically derived industry thresholds.

\end{enumerate}

\vspace{0.5cm}

These objectives are interdependent, collectively transforming static code analysis into a proactive, interpretable quality governance framework.
